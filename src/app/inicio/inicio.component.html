Bienvenidos a la API de pokemon

  • Te permite acceder a información detallada sobre todos los aspectos del universo Pokémon. Ya sea que estés construyendo una aplicación, un sitio web, o un proyecto personal.

  • Nuestra API tiene todo lo que necesitas:
      - Características PrincipalesPokémon
      - Información detallada sobre todos los Pokémon
      - Tipos, fortalezas y debilidades
  • Hemos utilizado la api de pokeapi.co/ la cual tiene todos los recursos que hay disponibles hasta el dia de hoy y actualizados recientemente de todos los juegos y peliculas
